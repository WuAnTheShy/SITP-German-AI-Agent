# 公共知识库：资料格式与准备要求

管理员在 **管理员工作台 → 公共知识库** 中上传的资料，会经系统**抽取文字 → 分段 → 向量化**，供师生在 **AI 对话中按语义检索**。  
请按下列要求准备资料，避免上传后显示 **失败** 或 **切片为 0**。

---

## 1. 支持的文件格式（仅此三种）

| 扩展名     | 说明                                                       |
| ---------- | ---------------------------------------------------------- |
| **`.pdf`** | 最常用；须为**可复制文字**的 PDF（见下文「PDF 特别说明」） |
| **`.txt`** | 纯文本，建议 **UTF-8** 编码                                |
| **`.md`**  | Markdown 文本，建议 **UTF-8** 编码                         |

其他格式（如 `.docx`、`.pptx`、图片、压缩包）**当前系统不接受**，需先导出或转换为上述格式。

---

## 2. PDF 特别说明（最容易踩坑）

系统会从 PDF 里**自动抽取文字层**，**不会对整页图片做 OCR（文字识别）**。

### ✅ 适合上传的 PDF

- 用 Word / LaTeX 等**直接导出**的 PDF
- 电子教材、带**可复制文字**的讲义
- 在 PDF 阅读器里**能选中、复制**出大段文字的文档

### ❌ 不适合或需先处理再上传

| 情况                                             | 处理方式                                                                                             |
| ------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| **扫描件、拍照、纯图片型 PDF**（页面上选不中字） | 先用 **OCR 软件**（如 Adobe、ABBYY、WPS 等）识别为可复制文字后，再导出 PDF 或改用 **`.txt` / `.md`** |
| 加密/禁止复制                                    | 换可编辑版本或向版权方索取可检索版本                                                                 |
| 版面极复杂且抽取乱码                             | 改为整理成 **`.md` / `.txt`** 再上传                                                                 |

**自检方法（30 秒）：** 用任意 PDF 阅读器打开文件，尝试**选中一段话并复制到记事本**。若复制不出正常文字，本系统**同样无法可靠索引**，需先 OCR 或换格式。

---

## 3. 文本类（`.txt` / `.md`）建议

- **编码：优先 UTF-8**，避免 GBK 等导致乱码。
- 内容可以是：词汇表、语法总结、课文、题库说明、学习笔记等**与学习相关的纯文本**。
- Markdown 中的标题、列表会按普通文本参与分段与检索，无需特殊排版。

---

## 4. 内容上与体量上的建议

- **主题**：与德语教学、课程、考试、词汇等相关的资料即可；**不要求**一定是「德语」写就，中文编写的讲义同样可以。
- **体量**：单文件过大时，**首次上传与建索引耗时会变长**，属正常现象；若长时间无响应，可联系管理员检查网络与后端。
- **分段方式（技术说明）**：系统会按固定长度将全文切成多段（约数百字符一段、段间有重叠），**不需要**组员手动切分。

---

## 5. 上传后如何确认「可用」

在管理员界面查看该条资料：

- **状态为 `ready`**，且 **切片数 > 0**：表示已成功入库，师生对话可检索。
- **状态为 `failed`**：请阅读界面上的**错误说明**；常见原因是 **PDF 抽不出文字**（扫描件），按上文第 2 节处理后可重新上传或点 **「重建索引」**（若已换源文件需重新上传）。

---

## 6. 简要 checklist（自检）

- [ ] 格式是否为 **`.pdf` / `.txt` / `.md`** 之一？
- [ ] 若是 PDF：**能否选中并复制文字**？
- [ ] 文本文件是否为 **UTF-8**？
- [ ] 是否避免仅含整页扫描图、无文字层的 PDF？

---

_说明内容依据当前系统实现整理；若后续产品增加格式（如 Word）或 OCR，以项目更新为准。_
