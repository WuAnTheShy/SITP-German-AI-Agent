# SITP German AI Agent 系统流程与架构

本模块通过 Mermaid 图表展示系统的核心业务流程与架构设计，帮助开发者快速理解系统逻辑。

---

## 1. 核心业务流程 (Core Workflows)

### 1.1 试卷生成与分发流程 (Exam Generation & Distribution)

```mermaid
sequenceDiagram
    participant T as 教师端 (React)
    participant B as 后端 (FastAPI)
    participant G as Gemini API
    participant D as 数据库 (PG)
    participant S as 学生端 (React)

    T->>B: 提交试卷配置 (题型数量, 策略)
    B->>G: 发送 Prompt (请求生成 JSON 格式题目)
    G-->>B: 返回结构化题目数据
    B->>D: 存储 Exam 记录 (ID, Content)
    B->>D: 为全班学生创建 ExamAssignment
    B-->>T: 返回成功提示
    S->>B: 查询待办任务
    B->>D: 读取 ExamAssignment
    D-->>B: 返回任务列表
    B-->>S: 展示新试卷
```

### 1.2 考试提交与评分流程 (Exam Submission & Scoring)

```mermaid
flowchart TD
    A[学生点击提交试卷] --> B{是否存在客观题?}
    B -- 是 --> C[后端自动对齐正确答案评分]
    B -- 否 --> D[跳过客观评分]
    C --> E[存储 Homework 记录]
    D --> E
    E --> F[更新学生总分与活跃度]
    F --> G[教师端可见待批改状态]
    G --> H[教师批改主观写作题]
    H --> I[生成最终成绩与 AI 诊断建议]
```

### 1.3 AI 情景对话交互流程 (AI Scenario Chat)

```mermaid
sequenceDiagram
    participant S as 学生 (React)
    participant B as 后端 (FastAPI)
    participant G as Gemini (Tutor)
    
    S->>B: 发送德语消息 (含 SessionID)
    B->>B: 注入系统提示词 (角色/难度/性格)
    B->>G: 组合 Context + History + NewMessage
    G-->>B: 返回德语回复 (带翻译与纠错)
    B-->>S: 前端流式/全量展示回复
    Note over S,G: AI 实时检测语法错误并显示在纠错栏
```
