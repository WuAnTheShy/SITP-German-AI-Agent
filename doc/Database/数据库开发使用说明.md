# SITP German AI Agent 数据库架构说明

> **核心框架**：PostgreSQL 16 + SQLAlchemy 2.0 (Mapped Column 语法)
> **当前表总数**：核心 ORM 表 33 张（见 `backend/models/entities.py`），另有知识库与 Trace 表（见 Alembic 迁移）

---

## 1. 核心业务表 (Core Business)

| 表名                      | 描述             | 关键字段                                                                                      |
| ------------------------- | ---------------- | --------------------------------------------------------------------------------------------- |
| `users`                   | 统一用户表       | `username`, `user_no`, `password_hash`, `role` (teacher/student/admin), `status`, `is_active` |
| `system_settings`         | 系统设置表       | `setting_key`, `setting_value`                                                                |
| `classes`                 | 班级信息表       | `class_code`, `class_name`, `teacher_user_id`                                                 |
| `class_teacher_relations` | 班级-教师关系    | `class_id`, `teacher_user_id`                                                                 |
| `class_student_relations` | 班级-学生关系    | `class_id`, `student_id`                                                                      |
| `teachers`                | 教师扩展表       | `user_id`                                                                                     |
| `students`                | 学生扩展信息表   | `uid` (学号), `user_id`, `status`, `overall_score`                                            |
| `student_abilities`       | 四维德语能力画像 | `listening`, `speaking`, `reading`, `writing`, `ai_diagnosis`                                 |

## 2. 教学任务与作业 (Instructional Tasks)

| 表名               | 描述                       | 关键字段                                                                                    |
| ------------------ | -------------------------- | ------------------------------------------------------------------------------------------- |
| `scenarios`        | 教师发布的 AI 情景配置     | `theme`, `difficulty`, `persona`, `goal_...`                                                |
| `scenario_pushes`  | 情景任务分发记录           | `scenario_id`, `student_id`, `push_status`                                                  |
| `exams`            | 智能试卷模版/实例          | `exam_code`, `strategy` (personalized/unified), `content` (JSONB)                           |
| `exam_assignments` | 试卷分发（支持个性化题目） | `exam_id`, `student_id`, `status`, `personalized_content` (JSONB)                           |
| `homeworks`        | 学生提交的作业/答卷        | `student_id`, `file_type` (text/audio/json_exam), `file_url`, `score`, `exam_assignment_id` |
| `homework_reviews` | 教师人工批改记录           | `homework_id`, `score`, `feedback`                                                          |

## 3. 学生学习工具 (Learning Tools)

| 表名                        | 描述                            | 关键字段                                                       |
| --------------------------- | ------------------------------- | -------------------------------------------------------------- |
| `chat_scenes`               | 自由对话或情景对话的主题字典    | `name`, `description`                                          |
| `chat_sessions`             | AI 对话会话（用于维持多轮记忆） | `student_id`, `scene_id`                                       |
| `chat_messages`             | 对话消息流水                    | `session_id`, `role` (user/assistant), `content`, `correction` |
| `teacher_chat_sessions`     | 教研助手会话                    | `user_id`, `title`                                             |
| `teacher_chat_messages`     | 教研助手消息                    | `session_id`, `role`, `content`                                |
| `vocabularies`              | 系统内置/AI 生成词表            | `german`, `chinese`, `level`, `topic`                          |
| `student_vocab_collections` | 学生生词记录                    | `student_id`, `vocab_id`                                       |
| `grammar_categories`        | 语法点分类 (如: 被动语态)       | `name`                                                         |
| `grammar_exercises`         | 语法练习题库                    | `category_id`, `question`, `correct_answer`                    |
| `grammar_submissions`       | 语法练习提交记录                | `student_id`, `exercise_id`, `is_correct`                      |
| `listening_materials`       | 听力素材库                      | `title`, `audio_url`, `script`                                 |
| `speaking_evaluations`      | 口语评估流水                    | `student_id`, `material_id`, `pronunciation_score` 等          |
| `writing_sessions`          | 写作辅助记录（纠错/生成）       | `student_id`, `session_type`, `result_json`                    |

## 4. 学情画像与辅助 (Learning Analytics)

| 表名                        | 描述             | 关键字段                                                   |
| --------------------------- | ---------------- | ---------------------------------------------------------- |
| `error_book_categories`     | 错题分类         | `name`                                                     |
| `error_book_entries`        | 聚合错题记录     | `student_id`, `question`, `correct_answer`, `is_mastered`  |
| `favorite_categories`       | 收藏分类         | `name`, `type`                                             |
| `favorites`                 | 学生收藏内容     | `student_id`, `content`, `translate`, `note`               |
| `learning_sessions`         | 学习时长埋点记录 | `student_id`, `module`, `duration_minutes`, `session_date` |
| `student_knowledge_mastery` | 知识点掌握度矩阵 | `student_id`, `knowledge_name`, `mastery_level`            |

## 5. 知识库与可观测性 (RAG & Observability)

| 表名           | 描述             | 关键字段                                                   |
| -------------- | ---------------- | ---------------------------------------------------------- |
| `kb_documents` | 知识库文档       | `title`, `source_path`, `scope`, `owner_user_id`, `status` |
| `kb_chunks`    | 知识库切片       | `document_id`, `chunk_index`, `embedding` (vector)         |
| `agent_traces` | Agent Trace 汇总 | `trace_id`, `role`, `user_id`, `session_id`, `rag_used`    |
| `agent_spans`  | Agent Span 明细  | `trace_id`, `span_type`, `span_name`, `duration_ms`        |

---

## 6. 开发分层架构

1. **模型层 (Models)**: `backend/models/entities.py` (SQLAlchemy 模型定义)。
2. **Schema层 (Schemas)**: `backend/schemas/entities.py` (Pydantic 请求/响应验证)。
3. **数据访问层 (CRUD)**: `backend/crud/repositories.py` (封装所有 DB 操作)。
4. **服务层 (API)**: `backend/routers/*.py` (路由与业务逻辑)，入口在 `backend/main.py`。

## 7. 数据库变更建议

- 对于复杂字段（如试卷题目、能力标签），统一使用 **JSONB** 存储。
- 关键查询（如 `user_no`、`username`、班级-学生/教师关系）已建立 **Unique Constraint**。
- 热路径索引已覆盖：`chat_messages(session_id, created_at)`、`learning_sessions(student_id, created_at)`、`homeworks(student_id, submitted_at)`、`error_book_entries(student_id, is_mastered)`。
